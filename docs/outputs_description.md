# Output descriptions

## Core manuscript pipeline

Typical outputs under the selected analysis output directory include:

- accession-level spectral feature tables
- genotype principal component summaries
- GWAS summary tables and candidate-gene annotations
- path analysis outputs
- in silico manifold perturbation outputs
- manuscript-supporting tables and figures

## Inter-seed post-GWAS pipeline

Typical outputs under the selected analysis output directory include:

- inter-seed GWAS summary tables
- candidate-gene interval summaries for inter-seed traits
- enriched annotations and descriptive tables

## support/support_tables.py

Typical outputs under the chosen summary output directory include:

- empirical LD decay summaries
- genotype PCA scree summaries
- chromosome 6 local interval summaries
- locus-level peak tables
- inter-seed overview tables
- workbook exports and CSV bundles for downstream inspection
